# Home Match

Home Match is a Python application to help users finding real estate according to their preferences.

## Features

* Create real estate listings
* Store listings in Vector DB
* Search listings based on preferences
* Personalized listing descriptions

## Prerequisites

* Python 3.7+
* LangChain
* ChromaDB
* OpenAI API

## Install and Run

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```
